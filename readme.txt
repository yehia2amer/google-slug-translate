Translate Chinese article aliases into English.

== Description ==

This plugin will automatically translate Chinese article aliases into English using Google.

== Installation ==

You can search for and install Google Slug Translate in the WordPress backend plugin directory. No extra settings.

== Frequently Asked Questions ==

= Why is there no "Google Translate" button when enabled? =

Please verify that your permalink structure contains a "%postname%" placeholder.

== Screenshots ==

1. Enter the title of the article and the translation will be automatically translated into English.
2. You can also manually translate by clicking the "Google Translate" button.
